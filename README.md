# Slither N' Slash

A standalone browser worm game with gradual turning, moving food targets, and a rechargeable boost bar.

Open `index.html` in a browser to play. Press `Play`, then hold Left/Right or A/D to turn gradually. Hold Up/W, Shift, or Space to spend boost, then release to recharge it. Longer snakes get a little more boost time. Down/S brakes, phones can steer by touch drag, and one new food target appears every 15 seconds.

## Files

- `index.html` - app shell
- `src/main.js` - game state, controls, drawing, scoring
- `src/styles.css` - responsive layout and game styling
- `assets/dark-rock-background.png` - generated dark rock background texture
- `audio/` - copied sound effects from the portfolio
